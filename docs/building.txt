The Mail Archive uses Bootstrap 5, the Sass-based front-end
component library.  Use the instructions below to compile
CSS if changes are required.

### frontend

* Install [Node](https://nodejs.org/en/download/package-manager/)
* Install [Yarn](https://yarnpkg.com/lang/en/docs/install/)
* cd <path to repo>frontend/
* yarn add package.json
* yarn build  (this will build once)
* OR yarn watch  (this will build every time a frontend file changes)
